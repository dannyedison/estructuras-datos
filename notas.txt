Los arreglos son un espacio de memoria que almacena multiples valores.
Los arreglos son un conjunto de datos  (cualquier tipo de datos)
Un arreglo se puede definir vacio: variable = []

    Tamaño -> Hace referencia a la cantidad de elementos dentro del arreglo
    Posición -> hace referencia a la cantidad de elementos -1 

Recorrido -> Se utiliza para buscar información
Llenado -> Se utiliza para alimentar el arreglo de información
